# Life Decision Simulator

This playful Python script guides the user through a mundane daily choice while treating the moment like a world-changing event. It uses nested if/elif/else statements, at least five input() prompts, and at least one Boolean operator to branch execution. Every possible path ends with a unique, ridiculously over-the-top outcome message.

The tone is intentionally dramatic and sarcastic — the goal is entertainment and to demonstrate branching logic and decision-tree structure in Python.

## Requirements (what the assignment requires)

Create a Python file named decision_tree.py.

Ask the user at least five questions using input().

Use nested if / elif / else statements (an if inside another if).

Use at least one Boolean operator (and, or, or not).

Every possible path must end with a unique, over-the-top "outcome" message.

Include a decision tree / flow diagram (see the Human Pseudocode section below).

## Installation

Make sure you have Python 3.8 or higher installed.

Clone this repository:

git clone <your-repo-url>
cd Life-Decision-Simulator

There are no third-party dependencies — just run the script with Python.
